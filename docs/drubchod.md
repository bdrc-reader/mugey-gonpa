# དགོན་པའི་སྒྲུབ་མཆོད་དང་ལྷ་སྲུང་སྐོར་གྱི་ངོ་སྤྲོད་མདོར་བསྡུས།

<br>

དམུ་དགེ་དགོན་པས་སྒྲུབ་མཆོད་ཆེ་ཆུང་གང་གི་སྐབས་སུ་ཡང་སྟོན་པ་ཐུབ་པའི་དབང་པོ་ནས་བརྒྱུད་པའི་རྒྱན་དྲུག་མཆོག་གཉིས་ཀྱིས་གཙོས་པའི་སྐྱེས་ཆེན་པཎ་གྲུབ་ཁག་དང་བཀའ་གདམས་ཀྱི་དགེ་བའི་བཤེས་གཉེན་རྣམས། རྗེ་བློ་བཟང་གྲགས་པའི་དཔལ་གཙོས་པའི་རིས་མེད་ཀྱི་སྐྱེས་ཆེན་རྣམས་དང་། ཡོངས་དགོས་སུ་༸རྒྱལ་བ་པན་ཆེན་རྣམ་གཉིས་དང་ཀུན་མཁྱེན་སྐུ་ཕྲེང་རིམ་བྱོན་སོགས་ཀྱི་བླ་རྒྱུད་གསོལ་འདེབས་རིགས་འདོན། ཡི་དམ་རྒྱུད་སྡེ་བཞི་ཡི་སྒྲུབ་མཆོད་ཡོད་མོད། དམིགས་བསལ་གྱིས་དགོས་པ་དང་རྟེན་འབྲེལ་གྱིས་དཔལ་གསང་བ་འདུས་པའི་བདག་འཇུག་བུམ་པ་དབང་དང་བཅས་པ་གྲྭ་ཚང་གཞུང་གིས་ཟླ་ཆོས་རེ་ལ་རེ་རེ་འཚོགས་བཞིན་ཡོད། ཡི་དམ་རྡོ་རྗེ་འཇིགས་བྱེད་དཔའ་བོ་གཅིག་པའི་ཆོ་ག་དང་དམ་ཅན་ཆོས་རྒྱལ་གྱི་ཆོ་ག་དྲུབ་ཅུ་བ་དང་བཅས་པ་ཉིན་རེར་ཆག་མེད་ཀྱིས་འདོན་བཞིན་ཡོད། 

ཆོས་སྐྱོང་གི་སྐོར་། གཙོ་བོ་སྐྱེས་བུ་གསུམ་གྱི་ཆོས་སྐྱོང་། མྱུར་མཛད་ཡེ་ཤེས་ཀྱི་མགོན་པོ་ཕྱག་དྲུག་པ་དང་དཔལ་ལྡན་ལྷ་མོ། དམ་ཅན་ཆོས་རྒྱལ། རྣམ་སྲས་རྟ་བདག་བརྒྱད། གནས་ཆང་་ཆོས་རྒྱལ་ཆེན་པོ། ཚངས་པ་དཀར་པོ་བཅས་ཡིན། མགོན་པོ་ཕྱག་དྲུགཔ་ནི་དགོན་པ་འདི་འི་དམིགས་བསལ་གྱིས་སྒྲུབ་མཆོད་དད་གུས་བྱེད། ཡུལ་མི་ཀུན་གྱིས་དམུ་དགེའི་མགོན་པོ་རང་བྱུང་ཞེས་འབོད། རྒྱུ་མཚན་རྗེ་རིན་པོ་ཆེས་ཁོང་གི་ཐུགས་སྲས་ཚ་ཁོ་ངག་དབང་གྲགས་པ་ལ་ཙནདན་གྱིས་བྱས་པའི་མགོན་
པོའི་སྐུ་བརྙན་ཞིག་གནང་ནས་འདི་དགོན་པ་དེའི་རྟེན་དུ་ཞོགས་དང་མ་འོངས་པར་བོད་ཁ་བ་ཅན་ན་སངས་རྒྱས་ཀྱི་བསྟན་པ་གནས་སའི་ཟོ་རྐུབ་ལྟ་བུ་ཞིག་ཡོང་གསུངས་ནས་ལུང་བསྟན་པ་རེད་ཅེས་དགོན་པའི་ཡིག་ཆ་་སོགས་ན་གསལ། དེའི་ཕྱིར་དགོན་པ་འདི་ཐོག་མར་བཏབ་པ་ནས་ད་བར་ཙན་དན་གྱི་མགོན་པོའི་སྐུ་བརྙན་དེ་དགོན་པའི་རྟེན་གྱི་གཙོ་བོ་བྱེད་བཞིན་ཡིན། རྨ་ཆུ་དང་མཛོད་དགེ། རྔ་བ། ཁྱུང་མཆུ། ཟུང་ཆུ། ཁྲོ་ཆུ་སོགས་ནས་ཡུལ་མི་ཧ་ཅང་མང་པོ་དམུ་དགེའི་མགོན་པོ་ལ་སྐབས་ཏུ་ཡོང་གི་ཡོད།


# སྒྲུབ་མཆོད་ཀྱི་སྐོར།

* དཔལ་གསང་བ་འདུས་པའི་བདག་འཇུག བུམ་པ་དབང་བཅས་པ།

* དཔལ་རྡོ་རྗེ་འཇིགས་བྱེད་ལྷ་བཅུ་གསུམ་མ།

* དཔལ་རྡོ་རྗེ་འཇིགས་བྱེད་དཔའ་བོ་གཅིག་པ།

* མགོན་ཆོས་ཀྱི་སྐོང་གསོལ་དང་དེ་དག་གི་མཆོད་དབྱངས་བཅས་པ།

* ཀུན་རིག་རྣམ་པར་སྣང་མཛད་ཀྱི་དབང་།

* ཀུན་རིག་རྣམ་པར་སྣང་མཛད་ཀྱི་ཁྲུས་ཆོག

* རྡོ་རྗེ་མི་འཁྲུག་པའི་བདག་འཇུག

* བླ་མ་མཆོད་པ།

* རིག་བྱེད་མ།

* སྒྲོལ་མའི་མཎྜལ་བཞི་ཆོག

* རྟ་མགྲིན་སྐོང་གསོལ།

* སྒྲོལ་མའི་རླུང་རྟ།

* རྣམ་སྲས་གཡང་འགུག

* སྒྲོལ་མའི་གཡང་འགུག

* དགྲ་བླའི་རླུང་རྟ།

# སྟོང་མཆོད་ཀྱི་སྐོར།


* རྗེ་སྟོང་མཆོད།

* སྨན་བླ་སྟོང་མཆོད།

* རྣམ་རྒྱལ་མའི་སྟོང་མཆོད།

* གནས་བརྟན་ཕྱག་མཆོད།

# གདུང་མཆོད་ཀྱི་སྐོར།


* རྗེ་རིན་པོ་ཆེའི་གདུང་མཆོད།

* ཚ་ཁོ་ངག་དབང་གྲགས་པའི་གདུང་མཆོད།

* ཀུན་མཁྱེན་སྐུ་ཕྲེང་རིམ་བྱོན་གྱི་གདུང་མཆོད།

* རྗེ་དམུ་དགེ་བསམ་གཏན་གྱི་གདུང་མཆོད།
